# NKVPN Moderator Bot Showcase

Public-safe showcase of a Telegram bot for support moderators. The private version lets moderators view, assign, reply to, and close user support tickets.

## Highlights

- Telegram moderation workflow.
- Ticket assignment and status updates.
- Shared support API with the admin panel.
- Safe command routing and action handling.

## Tech Stack

- Node.js, TypeScript.
- grammY Telegram framework.
- HTTP integration with support API.

## Included

- `docs/architecture.md` - bot and support API boundaries.
- `examples/ticket-routing/ticketRouting.ts` - safe routing example.

## Not Included

Telegram tokens, moderator identities, JWT/API tokens, support logs, chat IDs, and real ticket content are excluded.
